printf 里的类型转换

printf 里的类型转换，下面这里(unsigned char)data[i]，如不强制转换 会出出错，会当成有符号整数, 转成很大的十六进制字符串

void CLocationDlg::RevData(std::string &data) {
	wchar_t word[8] = {0};
	for(int i = 0; i < data.length(); i++) {
		swprintf_s(word, 8, L"%02X", (unsigned char)data[i]);
		word[3] = 0;
		m_com_string += word;
	}
}
