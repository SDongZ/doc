Git 中文问题

  1. Gitbash中显示中文：C:\Program Files\Git\etc\git-completion.bash 文件头添加 ：alias ls='ls --show-control-chars --color=auto'
  2. git log中显示中文：最新版本的GIT已经没有这个问题。只要LOG记录是UTF8格式的就没有问题。
  3. GIT LOG 编辑器设置: git config --global core.editor notepad 或者 完整路径：路径中的括号问题Program Files (x86)
  4. "C:\Program Files (x86)\Notepad++\notepad++.exe" 要写成 "C:/Program\ Files\ \(x86\)/Notepad++/notepad++.exe"
  5. "C:\Program Files\Notepad++\notepad++.exe" -multiInst -nosession 打开新窗口
  6. 设置"C:\Program Files\Notepad++\notepad++.exe" -multiInst -nosession后打开的临时文件不对
